## Overview

This repository contains **structured and modular implementations of core Machine Learning concepts**, built to strengthen both **theoretical understanding and practical ML engineering skills**.

The focus is on:
- understanding **model behavior**
- designing **reusable preprocessing pipelines**
- performing **model evaluation and comparison**
- improving **interpretability and explainability**

Rather than end-to-end applications, this repository emphasizes **clean ML workflows and fundamentals** that scale to real-world projects.

---

## Key ML Concepts Implemented

- Data processing and preprocessing pipelines

- Regression (linear & non-linear)
